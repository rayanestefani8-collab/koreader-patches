# KOReader TitleBar Hide Patch

Patch para o KOReader que oculta a barra de título no Gerenciador de Arquivos/Browser, permitindo que a grade de capas ocupe a tela inteira.

## ✨ Funcionalidades
* Remove a barra superior (título + menu) no FileManager.
* O espaço liberado é aproveitado pela grade de capas.
* **Correção:** Mantém os botões de fechar (X) visíveis em popups e menus de configuração.

## 🚀 Como Instalar
1. Baixe o arquivo `2-hide-titlebar-filebrowser.lua` deste repositório.
2. Conecte seu e-reader ao computador.
3. Copie o arquivo para a pasta `koreader/patches/` na memória do dispositivo.
4. Reinicie o KOReader.

## ⚠️ Notas
* A navegação entre pastas ainda funciona via gestos ou toque longo em espaços vazios.
* Testado em dispositivos Kindle com KOReader 2025.04+.
